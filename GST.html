GST Returns Overview

All registered businesses must file GST returns based on their business type. Here’s a quick breakdown:

  • GSTR-3B: Monthly filing for all registered non-composition businesses.
  • GSTR-1: Monthly filing if turnover exceeds ₹1.5 crores (or quarterly if below ₹1.5 crores).
  • Composition Holders: Must file quarterly returns.
